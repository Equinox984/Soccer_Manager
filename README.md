# Soccer Manager Engine ⚽
> CLI tool to explore Honduran football leagues, check upcoming matches, and test your soccer knowledge.

This Python program lets you navigate the Honduran football ecosystem through an interactive menu-driven interface. Browse First and Second Division teams, look up Wikipedia links, check upcoming fixtures, and challenge yourself with a trivia minigame.

## 🛠️ Technologies
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

## 🧠 How does it work?

The program offers three main modules accessible through an intuitive menu system:

### 🏟️ Soccer Information
- **Honduran Leagues & Teams:** Browse all teams in the First Division and the Ascenso League, organized by group
- **Finder:** Search for any team by name (case-insensitive) and get its league, group, and Wikipedia link

### 📅 Matches
- **First Division Matches:** View upcoming First Division fixtures with stadium and date details
- **Second Division Matches:** View upcoming Ascenso League fixtures

### 🧩 Trivia Minigame
- **Play:** Answer 5 random questions about Honduran football with multiple-choice options
- **Highest Score:** Track your best score for the current session

## 🚀 Installation and Use

**Requirement:** Have **Python 3.x** and **Git** installed on your system.

Clone the repository and run the script directly in your terminal:

1. **Clone the repository:**
```bash
git clone https://github.com/Equinox984/Soccer_Manager
cd Soccer_Manager/
```

2. **Run the program:**
```bash
python3 main.py
```

3. **Navigate the menu:**
   - Select options using numbers (1, 2, 3, 4)
   - Use the Finder to search teams by name
   - Test your knowledge with the trivia minigame
